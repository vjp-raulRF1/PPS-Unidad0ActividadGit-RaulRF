# Calculadora Python

Bienvenido a la documentación de la Calculadora Python.

## Descripción

Esta calculadora tiene una interfaz gráfica usando Tkinter.

## Uso

Ejecuta el script `calculator/gui.py` para iniciar la calculadora.

## Requisitos

- Python 3.x
- Tkinter
